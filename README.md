#wishPatterns
UML：
三角箭头实线，继承
三角箭头虚线，实现
普通箭头实线，关联关系，指向被拥有者。 代码体现：成员变量
普通箭头虚线：依赖关系，指向被使用者。代码体现：局部变量，方法参数或者对静态方法的调用
空心菱形实线，聚合关系，指向整体。代码体现：成员变量 （聚合：部分可以离开整体单独存在）
实心菱形实线，组合关系，指向整体。代码体现：成员变量（组合：部分不可以离开整体单独存在）

 各种关系的强弱顺序：

        泛化 = 实现 > 组合 > 聚合 > 关联 > 依赖
